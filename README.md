# Nachsorge-Assist

Eine webbasierte Anwendung zur Unterstützung der Nachsorge und Prävention von Erkrankungen.

## Features

- Eingabe von Symptomen oder Diagnosen
- Erkennung von Notfallsymptomen (rote Flaggen)
- Personalisierte Nachsorge- und Präventionsempfehlungen
- Responsive Design für alle Gerätegrößen
- Barrierefreie Benutzeroberfläche

## Technologien

- Frontend: React mit TypeScript und TailwindCSS
- Backend: Node.js mit Express und TypeScript
- API: RESTful

## Installation

### Voraussetzungen

- Node.js (v16 oder höher)
- npm (mit Node.js installiert)

### Backend einrichten

```bash
# In das Backend-Verzeichnis wechseln
cd backend

# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev
